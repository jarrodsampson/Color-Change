# Color ChangeJS


A new way to color. Designed by [planlodge](https://github.com/planlodge).


## Getting Started

 1. Download and include the color-change.js file.
 2. Pick your selector and apply colorChange method.

## Example

    $(".yourSelector").colorChange({color: blue});
